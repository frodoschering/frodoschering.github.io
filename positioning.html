Container rotation and position modifications

[Introduction]

Free rotation

When swiping on mobile the bottom sheet with modifications slides down and the container can be freely rotated. Clicking or swiping the panel up slide the bottom sheet up again.

Lock orientation

When the bottom sheet is active we can lock the container orientation in certain sections for better control over positioning modifications. This way the user can not accidentally rotate the container while position modifications. The user can still switch to free rotation mode at any time. When the bottom is made active again the container locks back into place.
